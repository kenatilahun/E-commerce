# Frontend Routes

## Main Mockup Routes

The electronics shop mockup now uses the main clean routes:

- `/`
- `/products`
- `/product`
- `/cart`
- `/checkout`

## Old Store Routes

The previous storefront pages were moved to simple backup routes:

- `/store-home`
- `/store-product/:id`
- `/store-cart`
- `/store-checkout`

## Other Existing Routes

These routes stay available as they were:

- `/login`
- `/register`
- `/profile`
- `/categories`
- `/category/:slug`
- `/myaccount/pro`
- `/admin`
